# Start Here

Copy this message into Codex, Claude Code, OpenCode, or another coding app.

Before sending it, replace `your-store.myshopify.com` with your Shopify store domain.

```text
I want you to set up my Shopify store assistant for me.

My Shopify store is:
your-store.myshopify.com

Please do everything you can from your own tools. Get the starter files from:
https://github.com/jordyhaasje/hazify-shopify-agent

Prepare the workspace, install anything missing, connect my store, set up safe access, choose the store permissions needed for products, orders, refunds, customers, inventory, discounts, draft orders, pages, metafields, and theme work, and check that everything works.

Please do not give me a list of technical setup steps. Do not ask me to create an app in Shopify myself. Use this project to do the connection work for me.

If Shopify opens a browser window, pause and tell me in plain language what I am approving and why. Wait for me to finish that browser step before you continue.

Do not ask me to paste passwords, access tokens, app secrets, or private keys into chat.

After setup, confirm what is working and ask me: "What would you like to do with your store?"
```
