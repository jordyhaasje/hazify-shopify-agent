# Getting Started

Start with `START_HERE.md`.

The normal entrypoint after cloning is:

```bash
npm install
npm start
```

Use the launcher to set up the workspace, connect a theme, enable Shopify data-agent access, run doctor checks, or start theme development.

Low-level commands such as `npm run setup`, `npm run setup:agent`, `npm run configure`, and `npm run auth:advanced` are advanced fallback tools.
