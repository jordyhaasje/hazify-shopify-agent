# Create A Section

Create a new Shopify section for my theme.

Requirements:

- Inspect my existing theme first.
- Match my current theme style, CSS variables, buttons, spacing, and responsive patterns.
- Create a merchant-editable section.
- Include section schema settings.
- Include blocks if they make the section easier to manage.
- Include presets so I can add the section in the theme editor.
- Use scoped CSS if needed.
- Consider snippets, assets, templates, locales, and blocks if needed.
- Preserve app block compatibility where relevant.
- Run Theme Check.
- Show changed files and explain what each file does.
- Do not push live.
