# Duplicate A Section

Duplicate an existing section so I can customize it separately.

Rules:

- Find the original section and all related snippets, assets, locales, blocks, and templates.
- Create a clearly named duplicate.
- Avoid breaking existing templates that use the original section.
- Update schema names and presets so the duplicate is easy to identify in the theme editor.
- Reuse existing theme styles where possible.
- Run Theme Check.
- Show changed files.
- Do not push live.
